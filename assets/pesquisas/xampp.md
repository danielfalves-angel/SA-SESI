Pesquisa de XAMPP

XAMPP é um acrônimo que representa quatro tecnologias: Apache, MySQL, PHP e PHPMyAdimin. Esse pacote de software livre oferece uma solução completa para criar e gerenciar servidores web locais.

As ferramentas mais utilizadas no XAMPP
Apache
Ele atua como um entregador virtual, recebendo solicitações do seu navegador (como Google Chrome ou Firefox) e enviando de volta as páginas, imagens e dados do site que você deseja acessar 

MySQL
Ele utiliza a linguagem SQL para armazenar, organizar e manipular dados estruturados. 

PHP
Ele opera no lado do servidor, processando dados dinâmicos e enviando o resultado final (como HTML) para o navegador do usuário.

phpMyAdmin
Ele substitui a necessidade de usar comandos SQL complexos no terminal, permitindo criar, editar, excluir tabelas e realizar backups via navegador. 

Para realizar a instalação basta baixar o instalador no site oficial, executar o arquivo com permissão de administrador, selecionar os componentes desejados e, por fim, gerenciar os serviços (Apache e MySQL) através do Painel de Controle para criar um servidor web local

O ambiente, seja ele o meio físico-ecológico ou a infraestrutura tecnológica/social, é a base para o desenvolvimento local. Ele garante os recursos essenciais e a qualidade de vida, servindo como alicerce para a economia, o bem-estar e a inovação comunitária
